# COT
COT (произносится как `кот`) -- это эксперимент, в котором генерируются картинки с котами и подписью "Бу! Испугался?" и заливаются в s3-хранилище с ограничением на размер бакета.

![plot](./demo.png)

## Run experiment
```
docker-compose up -d
```
